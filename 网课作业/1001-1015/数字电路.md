# 单项选择题 
```
第1题 下列电路属于组合逻辑电路的是：A 
A、全加器
B、寄存器
C、计数器
D、触发器
答案：A
第2题 全加器有（ ）个输入信号。 B
A、4
B、3
C、2
D、1
答案：B
第3题 将BCD代码翻译成十个对应的输出信号的电路有（ ）个输入端。 B
A、3
B、4
C、5
D、6
答案：B
第4题 同步触发器的同步信号为零时，现态为1，次态为： C
A、无法确定
B、0
C、1
D、以上说法都不对
答案：C
第5题 组合逻辑电路任意时刻的输出信号取决于： B
A、该时刻前的输入信号
B、该时刻的输入信号
C、该时刻后的输入信号
D、电路原来的状态
答案：B
第6题 当编码器 74LS147 的输入端 I1 、 I5 、 I6 、 I7 为低电平，其余输入端为高电平时，输出信号为：C
A、1110
B、1010
C、1001
D、1000
答案：C
第7题 在大多数情况下，对于译码器而言： A
A、其输入端数目少于输出端数目
B、其输入端数目多于输出端数目
C、其输入端数目与输出端数目几乎相同
答案：A
第8题 下列选项不能消除竞争冒险的是： C
A、接入滤波电容
B、引入选通脉冲
C、改变输入信号
D、修改逻辑设计
答案：C
多项选择题
第9题 选出触发器的触发方式： ABCD
	A、电平
	B、脉冲
	C、边沿
	D、以上选项都正确
答案：A|B|C|D
第10题 状态表化简的原理： ABC
	A、完备或覆盖性
	B、最小性
	C、封闭性
答案：A|B|C
第11题 以下电路属于组合逻辑电路的是： ABCD
	A、数据选择器
	B、只读存储器
	C、编码器
	D、比较器
答案：A|B|C|D
第12题 T触发器所需激励值取决于： AB
	A、现态
	B、次态的异或
	C、次态
	D、现态的异或
答案：A|B
```
