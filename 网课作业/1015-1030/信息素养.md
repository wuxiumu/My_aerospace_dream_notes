## 单项选择题
```
第1题 下面的描述中，正确的是：（ ）
	A、外存中的信息，可直接被CPU处理
	B、计算机中使用的汉字编码和码是一样的
	C、键盘是输入设备，显示器是输出设备
	D、操作系统是一种很重要的应用软件
	答案：C
第2题 ___的特点是信息量很大，索引数据库规模大，更新较快，因特网上新的或更新的页面常在短时间内被检索到，而过期的链接会及时第移去。
	A、多媒体型
	B、全文检索型
	C、目录型
	D、文摘型
	答案：B
第3题 国际标准书号的简称？（ ）
	A、ISBN
	B、CSBN
	C、ISSN
	D、ISRC
	答案：A
第4题 汉字系统中的汉字字库里存放的是：（ ）
	A、内码
	B、外码
	C、字模
	D、国际码
	答案：C
第5题 某单位的人事档案管理程序属于：（ ）
	A、系统程序
	B、系统软件
	C、应用软件
	D、目标程序
	答案：C
第6题 （ ）的特点是内容新颖、信息量大、出版周期短、传递信息快、传播面广、时效性强，能及时反映国内外各学科领域的发展动态,是科研人员了解科研动态、前沿信息和进展情况的首选文献。
	A、图书
	B、期刊
	C、专利
	D、内部资料
	答案：B
第7题 根据使用环境和对作业的处理方式，操作系统可分为批处理操作系统、分时操作系统、实时操作系统、网络操作系统和（ ） 。
	A、分布式操作系统
	B、联机操作系统
	C、集中式操作系统
	答案：A
```    
## 多项选择题
```
第8题 原语类型的类别包括:（ ）
	A、请求
	B、指示
	C、相应
	D、证实
	答案：A|B|C|D
第9题 我们可以从这几个方面来衡量服务器是否达到了其设计目的:（ ）
	A、可靠性
	B、可用性
	C、可扩展性
	D、易用性
	答案：A|B|C|D
第10题 路由器的作用：（ ）
	A、连接不同网络
	B、选择信息发送路径
	C、放大信号
	D、转换信号
	答案：A|B
第11题 硬盘的主要性能参数：（ ）
	A、容量
	B、转速
	C、平均寻道时间
	D、缓存
	答案：A|B|C|D
第12题 综合布线系统包括：（ ）
	A、工作区（终端）子系统
	B、水平布线子系统
	C、垂直干线子系统
	D、设备间子系统
	答案：A|B|C|D
第13题 综合布线的设计要点：（ ）
	A、符合规范化标准
	B、根据实际情况设计
	C、注意选材和布局
	D、降低投入
	答案：A|B|C
```
