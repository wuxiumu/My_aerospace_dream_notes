## 单项选择题
```
第1题 D触发器也可以构成寄存器电路，当输入端D为0时，再加一个移位正脉冲，D触发器输出端Q=（ ）。
	A、1
	B、0
	C、1
	D、10
	答案：B
第2题 已知输入A，B为或逻辑关系，当输出为0时，AB可能的值是：
	A、A=1,B=0
	B、A=1,B=1
	C、A=0,B=0
	D、A=0,B=1
	答案：C
第3题 一个4位移位寄存器可以构成最长计数器的长度是：
	A、8
	B、12
	C、15
	D、16
	答案：D
第4题 对于JK触发器，无论其原处何态，只要（ ）相同，则所需激励相同。
	A、现态
	B、次态
	C、输入
	D、输出
	答案：B
第5题 由三个触发器构成的五进制计数器有（ ）个无效状态。
	A、1
	B、2
	C、3
	D、4
	答案：C
第6题 具有脉冲鉴幅能力的触发器是：
	A、施密特触发器
	B、单稳态触发器
	C、集成单稳态触发器
	D、以上选项都正确
	答案：A
第7题 由555定时器接成的单稳态触发器当电路充电后停在（ ）的状态，三极管截止。
	A、Q=1
	B、Q=0
	C、Q=1或Q=0
	D、都不行
	答案：A
第8题 COMS门电路，由于其阈值电压约为电源电压的一半，因而：
	A、其噪声容限较TTL电路小
	B、抗干扰能力较TTL电路稍差
	C、其噪声容限较TTL电路大
	D、无法比较
	答案：C
```
## 多项选择题
```
第9题 有10进制计数器，要得到5进制计数器可采用（ ）实现。
	A、并行进位方式
	B、串行进位方式
	C、置数法
	D、置零法
	答案：C|D
第10题 在单稳态触发器的输入端触发电路中，可以采用基极触发电路，也可以采用集电极触发电路。根据有效触发脉冲的极性不同又有：
	A、正尖顶脉冲触发
	B、负尖顶脉冲触发
	C、无法确定
	答案：A|B
第11题 描述矩形脉冲的主要参数：
	A、脉冲幅度
	B、脉冲周期
	C、脉冲宽度
	D、上升时间
	答案：A|B|C|D
第12题 555定时器能方便的组成如下哪些器件：
	A、施密特触发器
	B、单稳态触发器
	C、多谐振荡器
	D、以上都可以
	答案：A|B|C|D
```
